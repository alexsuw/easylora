# Support

## Getting Help

- **Bug reports**: Open an issue using the
  [bug report template](https://github.com/alexsuw/easylora/issues/new?template=bug_report.yml).
  Run `easylora doctor` and include the output.
- **Feature requests**: Open an issue using the
  [feature request template](https://github.com/alexsuw/easylora/issues/new?template=feature_request.yml).
- **Questions and discussion**: Use
  [GitHub Discussions](https://github.com/alexsuw/easylora/discussions).
- **Documentation**: See the [docs site](https://alexsuw.github.io/easylora/)
  or the [README](README.md).

## Troubleshooting

Common issues and solutions are documented in the
[troubleshooting guide](https://alexsuw.github.io/easylora/troubleshooting/).

## Security

For security vulnerabilities, see [SECURITY.md](SECURITY.md).
Do not open public issues for security reports.
